Easily Make A Minecraft Server (Free)


Tutorial #1, posted 16/03/2018, edited 18/03/2018
NOTE: This will most likely only work on Windows.
I have been running a Minecraft server for over a year now, and I would have started earlier if it didn't seem so hard.
Luckily, it isn't hard. It is actually extremely easy.
Step 1: Download Java from here. Java is needed for server software to run.
Step 2: Download pre-made server files. I made one for you. Click here to get the files.
Step 3: Download Beacon by clicking here. Click "Beacon.exe".
Step 4: Run Beacon, then click on the Minecraft option. Once you have done that, click start. This will port forward your IP to the correct port for running a Minecraft server.
Step 5: Open the premade server file, and click on the file named "start". Don't worry about any errors that it shows in the file, usually the server will run fine despite these errors. (If you get any)
Step 6: Connect to the server using your public IP. Find your public IP here
If you want other people to join your server, give them the ip. You should only give your IP to people you trust.
